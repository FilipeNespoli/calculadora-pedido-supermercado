# Calculadora de Pedido de Supermercado

Projeto desenvolvido em JavaScript durante os estudos do livro
"Lógica de Programação e Algoritmos com JavaScript" (Novatec).

## Funcionalidades

- Cálculo do subtotal
- Desconto de 10%
- Cálculo do valor total
- Parcelamento em 3 vezes
- Formatação de valores monetários

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Conceitos praticados

- Variáveis
- Operadores aritméticos
- Conversão de tipos com Number()
- Eventos com addEventListener()
- Manipulação do DOM
- Entrada e saída de dados
